# Business Insights

## Overview

The Ecommerce Sales Analytics Dashboard was created to analyze sales performance, customer behavior, product trends, and payment preferences using PostgreSQL (NeonDB) and Power BI.

The dashboard provides an executive-level overview of business performance through interactive KPIs and visualizations.

---

# Key Performance Indicators

- **Total Revenue:** ₹30.45M
- **Total Orders:** 89,316
- **Total Customers:** 89,316
- **Average Order Value:** ₹340.90

---

# Revenue Analysis

- Revenue trends indicate sales performance across different months.
- Monthly analysis helps identify seasonal demand and sales fluctuations.
- The dashboard enables comparison of revenue over different time periods using calendar filters.

---

# Product Performance

- Product categories contribute differently to overall revenue.
- The dashboard highlights the highest-performing product categories.
- Average product price helps understand pricing patterns across products.

---

# Customer Analysis

- Customer orders are distributed across multiple Brazilian states.
- State-wise filtering allows detailed regional performance analysis.

---

# Payment Analysis

- Customers use multiple payment methods.
- Payment distribution provides insights into customer purchasing preferences.
- The dashboard makes it easy to identify the most frequently used payment method.

---

# Order Analysis

- Order status visualization provides an overview of the fulfillment process.
- The dashboard allows monitoring of delivered, shipped, cancelled, and other order statuses.

---

# Dashboard Capabilities

The dashboard enables users to:

- Monitor overall business performance.
- Track revenue trends over time.
- Analyze product category performance.
- Compare customer distribution across states.
- Understand payment method preferences.
- Monitor order status distribution.
- Filter data dynamically using slicers.

---

# Conclusion

This dashboard demonstrates an end-to-end Business Intelligence workflow, including:

- Data Cleaning using PostgreSQL (NeonDB)
- Relational Database Design
- Data Modeling
- DAX Measure Development
- Interactive Dashboard Development using Power BI
- Business Insight Generation
